Virtual Reality Wheelchair Rugby

A wheelchair-accessible sports game designed for Oculus and Steam VR! Using Unreal Engine 5, my team and I challenged ourselves to design a VR sports game that would be fun without needing to stand. To meet our goal, we enriched ourselves on how people interact with wheelchairs in competitive sports and real-life. That knowledge and best game design practices helped us create a game that is both fun inclusive!

8080 disassembler

A personal mission with the sole objective of learning. Emulation is something I have always been facinated with. The idea of software imitating an entire game console seemed like pure witchcraft.
I cannot think of a more humbling yet rewarding project. I've divided up the project with the first piece being to create a disassembler. A tool that reads a file and matches its instructions to an instructionset I defined. The next step is creating the CPU, its registers and a proper memory map for it.

Sudoku Solver + Generator

A self-imposed project to re-create my favorite pass-time. Backtracking turned out to be a difficult concept to understand. Thanks to a chat during a walk, I've decided to incorporate state tracking for potentially faster board generation.
